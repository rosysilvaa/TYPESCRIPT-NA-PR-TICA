
# 🚀 Projeto de Estudos: Node.js com TypeScript

Este repositório foi criado para documentar meu progresso no aprendizado de **Node.js** utilizando **TypeScript**. O foco inicial é entender a tipagem estática no ecossistema JavaScript, configuração de ambiente e a criação de scripts básicos.

## 🎯 Objetivos do Projeto
* Configuração do `tsconfig.json` para diferentes ambientes.
* Uso de tipos, interfaces e generics no Node.js.
* Manipulação de módulos e integração com bibliotecas externas.
* Implementação de boas práticas de desenvolvimento (ESLint/Prettier).

## 🛠️ Tecnologias e Ferramentas
* **Node.js**: Ambiente de execução.
* **TypeScript**: Linguagem base.
* **TS-Node-Dev**: Reinicialização automática do servidor e compilação em tempo real.

## 📁 Estrutura de Pastas
```text
├── src/           # Arquivos fonte (.ts)
├── dist/          # Código compilado para produção (.js)
├── @types/        # Definições de tipos customizados (se houver)
├── tsconfig.json  # Configurações do compilador TypeScript
└── package.json   # Dependências e scripts do projeto
```


## 📝 Lições Aprendidas
Durante o desenvolvimento deste projeto, explorei como o TypeScript ajuda na prevenção de erros comuns em tempo de execução e como configurar o compilador para garantir a compatibilidade do código com versões específicas do Node.js.
